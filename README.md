# SkillSphere 🚀

🏆 3rd Prize Winner – UI/UX Hackathon

## 📌 Problem Statement
Students preparing for placements use multiple platforms (LinkedIn, coding sites, mock interview tools), making it difficult to track overall readiness and progress.

## 💡 Solution
SkillSphere is a unified dashboard that helps students track and improve their placement readiness in one place.

## ✨ Features
- Skill tracking (Aptitude, Coding, Communication)
- Progress analytics dashboard
- Smart recommendations
- Clean and user-friendly UI/UX

## 🎯 UX Focus
- Smooth onboarding experience
- Minimal friction navigation
- Clear data visualization

## 🛠️ Tech Stack
- HTML, CSS, JavaScript

## 🏆 Achievement
Secured 3rd place in a UI/UX Hackathon
